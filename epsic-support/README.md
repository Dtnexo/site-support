## Installation

<br/>

<div align="center">

| | Etape | Description |
|:---:|:---|:---|
| <img src="https://img.shields.io/badge/1-0078d7?style=flat-square" /> | **Telecharger** | Clique sur le bouton ci-dessus pour telecharger le fichier ZIP |
